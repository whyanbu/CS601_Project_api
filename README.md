# Web API for My Personal Webpage
This is a minimal Web API for serving my personal webpage:  
https://whyanbu.github.io/CS601_Project/  

## Table of Contents
Features  
Installation  
Project Structure  

### Features
Written with Express JS and the data is stored in Google Firebase.  

### Installation
Clone the repository:

bash  
git clone https://github.com/whyanbu/CS601_Project_api.git  
cd CS601_Project_api
